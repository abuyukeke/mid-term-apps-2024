# Mid-Term Apps 2024
Base repository for Mid-Term Exams

## Ödev Kriterleri

* Sınıflandırmaya uygun açık kaynak depolardan veri seti kullanılacak.
* Bu veri setine uygun en az 3 sınıflandırma modeli uygulanacak.
* Train Set ve Test Set Accuracy değerleri karşılaştırılacak. 
*(Overfitting var mı? Underfitting var mı? Yorumlanacak.)*
* Test Set tahmin değerleri ile Test Set üzerinden elde edilecek Accuracy, Recall, Precision değerleri yorumlanacak.
*(SGDClassifier, Random Forest model kullanılacak.)*
* Thresholds modeli oluşturulmalı.
**Chapter 3** içindeki **Performance Measures** konusundaki yöntemler ile Precision/Recall Curve ve ROC Curve sonuçları yorumlanacak.
* Precision/Recall Curve dengesi için en uygun trashold değeri belirlenecek.

Değerlendirmeler ilgili kodun üst kısmına Text eklenerek yapılacak.
